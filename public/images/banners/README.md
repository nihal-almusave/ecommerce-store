# Banner Images

Place your banner images in this directory with the following names:

- `banner-1.jpg` - First banner image (handbags on wooden table)
- `banner-2.jpg` - Second banner image (woman holding three handbags)
- `banner-3.jpg` - Third banner image (woman with tote bag)

## Image Requirements

- Recommended size: 1920x700px or similar aspect ratio
- Format: JPG, PNG, or WebP
- File size: Optimize for web (under 500KB per image recommended)

## How to Add Images

1. Save your images with the names above
2. Place them in this directory: `/public/images/banners/`
3. The Hero component will automatically use them in the carousel
